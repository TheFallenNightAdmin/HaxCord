What it does

Built Different

HaxCord patches Discord at the Electron level — giving you full control without the overhead.

⚡
ASAR Patching
Injects directly into Discord's Electron app. No browser extension, no wrapper — the real deal.
🧩
Plugin API
Full before/after/instead hooks into any Discord internal function via the Webpack module system.
🎨
Theme Loader
Drop any .css file into the themes folder. Live toggle from settings. No restart required.
🔄
Auto-Updater
Checks GitHub releases on startup. One-click updates with toast notifications.
🛡️
Clean Uninstall
Backs up your original app.asar. Run the uninstaller and Discord is back to stock, no trace.
🖥️
Settings Panel
Fully integrated into Discord's own settings. Plugins, themes, and about — all in one place.

Get started

Install in 3 Commands

Close Discord first. Then run the following in your terminal.

# Clone the repo
git clone https://github.com/TheFallenNightAdmin/haxcord
Cloning into 'haxcord'...

# Install dependencies
cd haxcord && npm install
added 1 package in 0.8s

# Inject into Discord
npm run inject
[HaxCord] Found Discord at: C:\Users\...\AppData\Local\Discord
[HaxCord] Backed up original app.asar
[HaxCord] Injected successfully!
[HaxCord] Restart Discord to finish. ✓
// To uninstall: npm run uninject — restores original Discord with no trace.
